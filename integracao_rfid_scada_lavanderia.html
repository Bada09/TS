Pontos em azul = leitores RFID fixos. Cada passagem gera um evento de localização com timestamp — o centro de controle calcula o tempo em cada etapa e detecta gargalos.

Tabela item_tracking (relacional)
CREATE TABLE item_tracking ( tag_epc TEXT NOT NULL, checkpoint TEXT NOT NULL, arrived_at TIMESTAMPTZ NOT NULL, batch_id TEXT, client_id TEXT, machine_id TEXT, cycle_id UUID, -- FK p/ cycle_metrics sla_deadline TIMESTAMPTZ, sla_risk BOOLEAN DEFAULT false, PRIMARY KEY (tag_epc, checkpoint, arrived_at) );
Integração SCADA — protocolos e mapeamento
OPC-UA preferido
Protocolo moderno com modelo de dados orientado a objetos. Suportado por PLCs Siemens, Rockwell, Schneider. Permite subscrição de variáveis — o servidor só envia quando há mudança (push), sem polling.
Siemens S7-1500Allen-BradleyBeckhoff
Modbus TCP legado
Protocolo de polling — o sistema pergunta ao PLC a cada N segundos. Simples e universal, mas sem push nativo. Usar quando o equipamento não suporta OPC-UA. Polling a cada 5–10s é suficiente para parâmetros de processo.
equipamentos antigosinversoresmedidores
MQTT IoT nativo
Ideal para sensores independentes (pH, turbidez, dosagem). Publish/subscribe com broker central (Mosquitto ou EMQX). Leve, tolerante a conexões instáveis — recomendado para sensores em campo via Wi-Fi ou 4G.

Lógica de correlação RFID + SCADA
Associação peça → ciclo
Quando o RFID lê um tag na "entrada_lavagem", o sistema consulta qual máquina está em ciclo ativo naquele ponto físico (via SCADA). O cycle_id é preenchido automaticamente no item_tracking — sem digitação manual.
Problema comum: lote dividido entre duas máquinas. Solução: o reader de entrada cobre a faixa de uma única máquina — layout físico resolve o problema lógico.
Detecção de peça perdida
Se um tag aparece na expedição sem ter passado por "entrada_lavagem" ou "saída_secagem", o sistema gera alerta de rastreabilidade quebrada. Se um tag não aparece na expedição 4h após entrar na lavagem, é marcado como "localização desconhecida" e aciona busca ativa.
-- peças sem checkpoint de saída há mais de 4h SELECT tag_epc, client_id, arrived_at FROM item_tracking WHERE checkpoint = 'entrada_lavagem' AND arrived_at < NOW() - INTERVAL '4h' AND NOT EXISTS ( SELECT 1 FROM item_tracking t2 WHERE t2.tag_epc = item_tracking.tag_epc AND t2.checkpoint = 'saida_secagem' );
